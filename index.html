Навыки

За большой промежуток времени я развивал и практиковал множества различных навыков.

HTML

HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — стандартизированный язык гипертекстовой разметки документов для просмотра веб-страниц в браузере.

CSS

CSS (от англ. Cascading Style Sheets «каскадные таблицы стилей») — формальный язык декорирования и описания внешнего вида документа (веб-страницы), написанного с использованием языка разметки (чаще всего HTML или XHTML).

JavaScript

JavaScript (аббр. JS) — мультипарадигменный язык программирования. Поддерживает объектно-ориентированный, императивный и функциональный стили. Является реализацией спецификации ECMAScript (стандарт ECMA-262).

Bootstrap

Свободный набор инструментов для создания сайтов и веб-приложений. Включает в себя HTML- и CSS-шаблоны оформления для типографики, веб-форм, кнопок, меток, блоков навигации и прочих компонентов веб-интерфейса, включая JavaScript-расширения.

PHP

PHP (от англ. Hypertext Preprocessor) — это мультипарадигмальный, объектно-ориентированный, императивный язык программирования, появившийся в 1995 году.

SASS

Наиболее развитое и стабильное расширение CSS профессионального уровня.

ADOBE

С помощью этого набора программ можно выполнять множество задач, связанных с творчеством, например: создавать и улучшать векторную и растровую графику на ПК, смартфонах и iPad, в том числе с помощью сотен тысяч готовых шаблонов; обрабатывать фотографии на любых устройствах; выполнять дизайн и верстку для печати и цифровых медиа; монтировать видеоролики и фильмы; создавать 2D- и 3D-анимацию и визуальные эффекты; разрабатывать прототипы и дизайн интерфейсов для сайтов и мобильных приложений; редактировать и создавать PDF-файлы; пользоваться генеративным ИИ для различных задач, связанных с графикой
